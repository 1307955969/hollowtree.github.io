2.DOM

笔记

通过ID获取元素: document.getElementById(“id”)

innerHTML 属性: Object.innerHTML

改变 HTML 样式: Object.style.property=new style;

  • backgroundColor
  • height
  • width
  • color
  • font
  • fontFamily
  • fontSize

    • 显示和隐藏（display属性）: Object.style.display = value

    控制类名（className 属性）: object.className = classname
